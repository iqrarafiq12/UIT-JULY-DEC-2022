3) The new Array() Constructor

The Array() constructor creates Array objects. You can declare an array with the "new" keyword to instantiate the array in memory.
Here’s how you can declare new Array() constructor:
let x = new Array(); - an empty array
let x = new Array(10,20,30); - three elements in the array: 10,20,30
let x = new Array(10); - ten empty elements in array: ,,,,,,,,,
let x = new Array('10'); - an array with 1 element: ‘10’
